<hr />

curved or block.

## state

<hr />

States on f-div are used to communicate purpose and its connotation. For example, a red color
connotes danger, whereas a green color connotes success and so on.

<p class="note-section">
  Note: In case of nested f-div, all children f-div inherit the state from
  parent f-div until specified.
</p>

## width

<hr />

Width of a layout can be defined in 3 ways:

- Fill container: width is set to the parent container’s width.
- Hug contents: width is a sum of the width of all elements inside it and left and right
  padding.
- Fixed: width is set to the numerical value passed to the width property.
