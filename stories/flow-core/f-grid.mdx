# f-grid

<br />

<br />

## Overview

<f-divider />

f-grid is a wrapper component to arrange data as a grid of cells. The f-divs wrapped inside f-grid are called cells.

### min-cell-width

<f-divider />

Defines the minimum value of width of each cell. A cell can not be smaller than this in any viewport.

<p className="note-section">Note: The min-cell-width can not take value in fraction or percent.</p>

### max-cell-width

<f-divider />

Defines the maximum width of each cell.

<p className="note-section">
	Note: max-cell-width accept number, percentage, or fraction as values.
</p>

### gap

<f-divider />

Defines the gap between the cells. By default it is set to small.

<p className="note-section">Note: gap property defines horizontal as well as vertical gap.</p>
